# 🎮 멀티플랫폼(크로스플레이) 게임 이용자 수익성 분석

## 1. 프로젝트 개요
본 프로젝트는 게임 이용자 실태조사 데이터를 활용하여  
**멀티플랫폼(PC–모바일) 크로스플레이 유저가 단일 플랫폼 유저 대비
더 높은 수익성을 보이는지**를 분석한 데이터 분석 프로젝트입니다.

## 2. 분석 배경 및 목적
- 최근 게임 산업에서 멀티플랫폼 전략의 중요성 증가
- 컴투스와 같은 모바일·PC 기반 게임사의 전략적 의사결정 지원
- 단순 평균 비교가 아닌, 연령·이용 빈도·결제 경험을 통제한 분석 수행

## 3. 데이터 안내 (중요)
본 프로젝트는 **한국콘텐츠진흥원 공공데이터**를 기반으로 분석되었습니다.  
저작권 및 이용 약관에 따라 **원본 데이터는 GitHub에 포함하지 않았습니다.**

대신,
- 분석 로직
- 전처리 방식
- 지표 설계
- 통계 검정 방법  
을 중심으로 분석 역량을 확인할 수 있도록 구성했습니다.

## 4. 분석 방법 요약
- 크로스플레이 정의: PC·모바일 Top3 게임 응답 간 교집합 여부
- 주요 지표:
  - 지표 A: PC+모바일 인게임 결제액
  - 지표 B: 전체 게임 지출액
- 통계 기법:
  - Mann–Whitney U 검정
  - 카이제곱 검정
- 통제 분석:
  - 연령대별 비교
  - 이용 빈도 통제 후 비교
  - 결제 경험 여부 분석

## 5. 핵심 분석 결과 요약
- 크로스플레이 유저는 단일 플랫폼 유저 대비 평균·중앙값 모두에서 지출이 높음
- 동일 이용 빈도 그룹 내에서도 크로스플레이 유저의 지출 우위 유지
- 결제 경험률 역시 크로스플레이 유저가 통계적으로 유의하게 높음
- RPG 장르(Top3 포함) 유저에서 수익성 상승 효과가 특히 큼
