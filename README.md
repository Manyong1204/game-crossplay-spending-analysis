# PC–모바일 크로스플레이 유저의 결제 행동 분석  
### 2024 게임 이용자 실태조사 기반 데이터 분석 프로젝트

---

## 📌 프로젝트 개요

본 프로젝트는 **PC–모바일 크로스플레이 여부가 유저의 게임 지출 행동에 어떤 영향을 미치는지**를 분석하기 위해 수행되었습니다.  
특히 단순 평균 비교를 넘어,

- 연령대별 차이
- 이용 빈도 통제 후 비교
- 결제 경험 여부
- 장르(RPG) 시너지 효과

등을 단계적으로 검증하여, **“크로스플레이 유저는 왜 더 높은 가치를 가지는가?”** 에 대한 실무적 인사이트를 도출하는 것을 목표로 합니다.

---

## 🎯 분석 목적 (Research Questions)

1. 크로스플레이 유저는 일반 유저보다 실제로 더 많은 금액을 지출하는가?
2. 이 차이는 **연령대가 달라도 유지되는가?**
3. 단순히 게임을 많이 해서 그런 것인가?  
   → **PC/모바일 이용 빈도를 통제해도 차이가 존재하는가?**
4. 크로스플레이는 **결제 경험 자체를 증가시키는가?**
5. RPG 장르에서 크로스플레이 시너지 효과가 더 크게 나타나는가?

---

## 📊 데이터 설명

- **데이터 출처**: 2024 게임 이용자 실태조사 (한국콘텐츠진흥원의 공공 조사 데이터를 제공받아 활용하였음)
- **분석 단위**: 개인(응답자) 단위
- **주요 특징**
  - PC / 모바일 게임 이용 및 결제 정보 포함
  - 게임 장르 Top3 응답
  - 이용 빈도, 연령대, 성별 등 인구통계 정보 포함

> ⚠️ 콘솔 게임은 주관식 게임명 데이터 부재로 인해  
> **타이틀 기반 크로스플레이 분석에서는 제외**하였습니다.

## 🧩 주요 변수 정의

### 1️⃣ 크로스플레이 여부 (`is_cross_play`)
- PC와 모바일에서 **동일한 게임 타이틀**을 플레이한 경우 `True`
- 각 플랫폼 Top3 게임 응답을 코드북으로 매핑하여 교집합 존재 여부로 판단

### 2️⃣ 핵심 지표

| 지표명 | 설명 |
|------|------|
| Indicator A | PC + 모바일 인게임 결제액 |
| Indicator B | 유저 총 게임 지출액 (PC + 모바일 + 콘솔) |
| 결제 경험 여부 | Indicator B > 0 여부 |

### 3️⃣ 장르 분류
- PC + 모바일 **Top3 장르 중 하나라도 RPG 포함 시 ‘RPG(Top3 포함)’**
- 그 외는 ‘기타’

## 🔍 분석 방법

- **기초 통계**: 표본수(n), 평균(mean), 중앙값(median)
- **집단 비교 검정**
  - 지출액: Mann–Whitney U test (비정규·롱테일 분포 고려)
  - 결제 경험 여부: Chi-square test
- **통제 분석**
  - 연령대별 비교
  - PC/모바일 이용 빈도 구간화(Low / Mid / High) 후 비교

---

## 📈 주요 분석 결과 요약

### ✅ 1. 크로스플레이 유저는 더 많은 금액을 지출
- 평균·중앙값 모두에서 크로스플레이 유저가 일반 유저보다 높음
- 통계적으로 유의미한 차이 확인

### ✅ 2. 연령대가 달라도 경향은 유지
- 모든 연령대에서 크로스플레이 유저의 지출이 더 높음
- 특히 **30대에서 격차가 가장 크게 나타남**

### ✅ 3. 이용 빈도를 통제해도 차이 존재
- 같은 이용 빈도 그룹(Low/Mid/High) 내에서도  
  크로스플레이 유저의 평균 지출이 더 높음
- 단순 ‘헤비 유저 효과’만으로 설명 불가

### ✅ 4. 결제 경험 자체를 증가시킴
- 크로스플레이 유저의 결제 경험률이 더 높음
- 카이제곱 검정 결과 통계적으로 유의

### ✅ 5. RPG 장르에서 시너지 효과 강화
- PC·모바일 Top3 내 RPG 포함 유저는  
  크로스플레이 시 지출 규모가 더 큼
- RPG 중심 멀티플랫폼 전략의 유효성 시사

---

## 💡 실무적 시사점

- 크로스플레이 유저는 **단순 활성 유저가 아닌 고가치 유저**
- 멀티플랫폼 환경은
  - 결제 진입 장벽을 낮추고
  - 장기적인 지출 확장을 유도
- 특히 **RPG 장르에서는 크로스플레이 지원이 핵심 전략 요소**
